# High-Demand, High-Paying AI Careers (2025–2026)

## Overview

This project analyzes the AI job market using 1,500 AI-related job postings from 2025–2026.

The objective is to identify AI career paths that provide the strongest combination of salary, demand, and long-term growth opportunities.

## Research Question

Which AI career paths offer the strongest combination of salary and market demand in the 2025–2026 job market?

## Tools

* Pandas
* Matplotlib

## Planned Analysis
